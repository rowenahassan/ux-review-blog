# UX Review Blog

A responsive blog landing page built using HTML and CSS with Flexbox layout.

## Features

- Fully responsive design (desktop, tablet, mobile)
- Flexbox-based layout
- Clean and modern UI
- Sticky sidebar
- Mobile navigation menu

## Technologies Used

- HTML5
- CSS3
- Flexbox

## Preview

You can check the responsive behavior by resizing the browser window or using Chrome DevTools.
